# SD Dragons Friday Pitch Report

Live dashboard for the SD Dragons Girls 2012 U13 team in the
California Soccer League — Friday Youth U13 (2013-2014) division.

**Live URL:** _set this after enabling GitHub Pages_ — typically
`https://<your-username>.github.io/dragons-dashboard/`

## What's in here

- `index.html` — the entire dashboard, single self-contained file.
  Includes the team crest as embedded base64, all data, all charts.
  No external dependencies except Google Fonts.
- `README.md` — this file.
- `.gitignore` — keeps editor cruft out.

## How to update the dashboard

The dashboard is generated by Claude based on data scraped from
`californiasoccerleague.com`. To refresh it:

1. Ask Claude to regenerate the dashboard with current data.
2. Download the new `index.html` from the chat output.
3. In this repo on GitHub:
   - Click `index.html`
   - Click the pencil icon (Edit)
   - Or drag-and-drop a new `index.html` over the file list ("Add file" → "Upload files")
   - Commit the change
4. GitHub Pages re-deploys automatically within ~60 seconds.

The footer of the dashboard shows a build hash and timestamp so you can
verify which version is live.

## Enabling GitHub Pages (one-time setup)

1. Repo → **Settings** → **Pages** (left sidebar)
2. **Source:** "Deploy from a branch"
3. **Branch:** `main`, folder `/ (root)`
4. Click **Save**
5. Wait ~60 seconds, then your URL appears at the top of the Pages settings

## Optional improvements

- **Custom domain:** Settings → Pages → Custom domain. Requires a domain
  you own and a CNAME DNS record pointing at `<username>.github.io`.
- **HTTPS:** Enabled automatically for `github.io` URLs and for custom
  domains after DNS propagates (~minutes to a few hours).
- **Auto-update from CSL:** A GitHub Action could scrape the league site
  on a schedule and commit a new `data.json`, then a build step bakes
  it into `index.html`. Not currently set up.

## Data source

All data comes from the public CSL standings page:
https://www.californiasoccerleague.com/leagues/friday-youth-u13-2013-2014/

Match logs cover only the most recent ~5 weeks (the league site doesn't
archive earlier games). Standings totals reflect the full season.
